# VaultCLI

CLI-приложение на Go для безопасного хранения секретов (логины, пароли, заметки) с локальным JSON-хранилищем.

---

## Текущий функционал

- Создана доменная модель записи (Entry)
- Реализована валидация данных
- Добавлены утилиты:
  - генерация ID
  - получение времени в едином формате
- Определён интерфейс хранилища (`Storage`)
- Реализовано файловое хранилище (`FileStorage`)
  - чтение из JSON
  - запись в JSON
  - обработка случая отсутствия файла

---

## Архитектура

Проект строится с разделением на слои:

- `entity` — доменные модели
- `utils` — вспомогательные функции
- `storage` — работа с данными (интерфейс + реализация)
- `service` — бизнес-логика (в разработке)
- `cli` — обработка команд (в разработке)

---

## В разработке

Планируется реализовать:

- сервисный слой (CRUD операции)
- CLI-интерфейс (команды: add, list, get, delete, update, search)
- шифрование данных
- работа с `.env`
- HTTP-функциональность (проверка URL)
- unit-тесты
- import/export данных
